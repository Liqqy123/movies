Цель проекта
Данный проект представляет собой консольное приложение для управления каталогом фильмов. Оно позволяет пользователям:
Просматривать список фильмов.
Добавлять новые фильмы.
Отмечать фильмы как просмотренные.
Искать фильмы по году выпуска.

Способ запуска программы
Для запуска приложения выполните следующие шаги:

Убедитесь, что у вас установлен Python версии 3.x.
Клонируйте репозиторий на ваш компьютер:
